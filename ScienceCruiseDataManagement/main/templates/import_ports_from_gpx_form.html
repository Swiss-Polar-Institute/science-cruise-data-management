Steps to export/import

In OpenCPN export the route using GPX format and upload the file here.

To export:

  • Go to OpenCPN
  • Press the button
  • Select the route to export
  • Press "Export select" button and save it into a file
To import it here:
  • Press the button on the top of the page and choose the file that has just been exported
  • Press "Upload"
What will happen?
  • Will update existing Stations (based on their names) with the new locations
  • Will create the new stations
  • The Stations names are the names on the OpenCPN, the leg is the active leg (the last leg started based on the current date)
